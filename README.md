# Alex Welcing - AI Product Manager Portfolio

A modern, performant portfolio website showcasing AI product management expertise, technical leadership, and thought leadership in intelligent systems.

## 🚀 Overview

Professional portfolio site built with Next.js, optimized for SEO and designed to attract recruiters and potential employers in the AI/ML product management space.

### Key Features

- **SEO-Optimized**: Comprehensive meta tags, structured data, and social media optimization
- **Performance-First**: Static generation, optimized assets, and minimal dependencies
- **Mobile Responsive**: Fully responsive design that works on all devices
- **Content Management**: Article system for sharing insights on AI product management
- **Professional Design**: Clean, modern interface focused on clarity and impact

## 🛠 Tech Stack

- **Framework**: Next.js 15 with React 19
- **Styling**: Tailwind CSS with custom design system
- **Content**: MDX for articles with gray-matter frontmatter
- **Database**: Supabase for article storage and analytics
- **Deployment**: Optimized for Vercel
- **SEO**: next-sitemap for automatic sitemap generation

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/alexwelcing/NextDocsSearch.git

# Navigate to the project directory
cd NextDocsSearch

# Install dependencies
npm install --legacy-peer-deps

# Run the development server
npm run dev
```

The site will be available at [http://localhost:3000](http://localhost:3000).

## 🔧 Development
