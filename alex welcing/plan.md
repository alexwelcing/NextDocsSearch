# Plan: Consequences — Satirical YA Rewrite + Authorial Voice System

Date: 2026-06-16 PDT

## Stage 1: Market Research & Voice Architecture
- Research YA satirical tone, Scholastic circuit expectations, character-author construction
- Research how to build repeatable authorial voice frameworks for different author+intent combos
- Output: Research brief + voice architecture blueprint

## Stage 2: Source Material Intake
- User provides the "Consequences" foundation story
- Output: Annotated source, word count baseline, structural map

## Stage 3: Authorial Voice Construction
- Build the persona: "sci-fi Hemingway, Brown bookish delight, mercurial temperament, comic book action sensibility, loves love/fight scenes, lifted by those above and beyond"
- Define voice rules, diction register, sentence architecture, humor mechanics
- Output: Authorial voice profile (repeatable template)

## Stage 4: Satirical Rewrite Execution
- Cut ~1/3 word count
- Age down for YA Scholastic circuit
- Add gauche high-society literary references (humor)
- Refuse sanguine conclusions — keep it sharp, unsentimental
- Preserve story arc but change emotional valence
- Target: punchier, satirical, gratuity within YA bounds
- Output: Rewritten manuscript in markdown

## Stage 5: Validation
- Word count check: is it ~2/3 of original?
- Tone check: satirical? high-society literary refs? gauche?
- YA appropriateness check: no explicit content, but edge-of-gratuity selling points preserved
- Voice consistency check: does it sound like our constructed author?

## Repeatability Design
- Package voice construction as a template system so future author+intent combos can be instantiated quickly
- Each new author = new persona profile + tone intent + target demo
- This becomes a skill or reusable framework under the project
